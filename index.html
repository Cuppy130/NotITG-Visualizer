NotITG Editor

NotITG Editor is a web-based editor for NotITG modfiles.

Features

  • Import and export NotITG modfiles
  • Visual editor for modfiles
  • Support for all NotITG modfile features

Usage

Drag and drop a NotITG modfile onto the editor to load it, it must be a folder that contains the Mirin Template

Click the Export button to download the modfile(s?)

Source Code

Source code is available on GitHub

Control

  • Mouse wheel/Up/Down: Move forwards and backwords in the song position by the selected measure.
  • Left/Right: Change the selected measure to a greater or lesser value.
  • Spacebar: Play song at the current time.
